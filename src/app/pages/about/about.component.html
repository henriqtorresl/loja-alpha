Sobre o Projeto

O "Loja Alpha" é uma aplicação web desenvolvida como parte de um processo seletivo para a vaga de Desenvolvedor(a) Júnior - Front End. O principal objetivo do projeto era criar uma interface intuitiva e responsiva para o gerenciamento de produtos e a autenticação de usuários, com um sistema completo de CRUD que permite a criação, visualização, atualização e exclusão de produtos. Desenvolvida em Angular, a aplicação integra-se à API fornecida pela empresa, proporcionando uma experiência segura e eficiente.
